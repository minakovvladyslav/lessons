  /* селектори
  
    >-знак бідьше задає стилі усім div першого рівня(а не лише першому div) в котейнері .section 

  .section > div {
  color:red
  }

  ~ - знак тільда задає стилі елементам пілся контейнера(обьекта  які не лежать в цьому контейнері),того свмого рівня вкладенності

  .container ~ p {
    color: red;
  }

  + задае стилі обьектам які лежать відразу після контейнера(не в ньому)

  .container + p {
    font-size: 25px;  
  }


  якщо ми задаемо стилі в HTML - <div style="color:red;">- то це має найвіщий пріорітет

  */

  для backgraund-position  - задаються відступи спершу здіва на право,а потім зверху в низ
  також можно розміщувати словами( right bottom)

  якщо задаємо background у процентах,то ми зміщуемо вісоту відностно висоти обьекта або вьюпорта та таким чином ширину,але ми зміщуємо його цетнр а не кути
  
  background-size:contain  - зображення маштабуеться,але не обрізяється
  background-size: cover  -зображення повністю накриває обьект,але обрізається(часто використовується з backgraund-position: center)
  background-clip: padding-box, якщо боксу задан border то ця властивість робить border як outline
  background-clip: text - встановлення фону для тексту
  backgroun: url('...') center/200px no-repeat scroll  - center/200px зпершу розташування обьекта / його розмір
