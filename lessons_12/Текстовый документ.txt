--трансформації--

transform- не працює з строковими елементами!(p,span,h1...)

translate: рухає центральну точку обьєкту(transform: translate( 20px, 30px))
transform: можна задати декілька трансформацій
translate: можна задати лише цю властивість(окремо для css-translate: -70px, 20px)

transform: scale(2)-відповідає за маштабування обьекту де (1) це початковий розмір обьекту
transform: scale(2, 1)- розмір по X та Y
transform: scale(-2, 1) - буде дзеркальне відображення

transform: skewX(-10deg)
transform: skewY(10deg)
transform: skew(-10deg, 0) -диформуть,згинають обьект

transform: translate() scale() rotate() skew()

transform-origin: top right - задає з якоЇ точки буде проволитись трансформація(позиціонування схоже на background-position)


perspective: 100px-буде близько до екрану(до нас) 1000px- буде далеко від екрану
perspective-origin:позиціонування анімацуії (задаються батьківському елементу!!!!!!!!!!)

Для 3D - трансформацій обов'язково задаємо perspective!
transfporm: rotate3d(1,2,1,deg); - де x,y,z,та градуси

transform-style: preserve-3d робить анімацю 3Д , за замовченням flat

backface-visibility: hidden- скриває зворотню сторону анімованого обьекту(видно коли 180deg)

--фнімації--
transition-duratuion: за який промыжок часу здійсниться анімація

.text {
  color: #fff;
  transition-duration: 0.3s;
}

.text:hover {
  color: red;
  transition-duration: 0.8s;
} - якщо я наведу курсор на обьект,то анімація займе 0.8секунд, якщо відведу то анімація займе 0.3секунд

transition-property: до якого обьекту застосовується анімація- за замовченням (all)

transition-duration: 0.3s, 0.8s;
transition-property: color,width- color занімується за 0.3с ,а width- за 0.8с

transition-delay: -на скільки анімація затримається

transition-duration: 0.3s, 0.8s;
transition-property: color,width;
transition-delay: 0 , 1s;


transition-timing-function: задає стиль,тобто сценарій для нашої анімації

transition-timing-function: linear - робить так щоб динаміка анімації була з аочатку до кінця та в заоротньому напрямку однаковую
transition-timing-function: steps(20, start); - робить анімацію за кроками(буде в 20 кроків)
animation-iteration-count: скільки разів повториться анімація, infinite- буде повторюватися завжди
